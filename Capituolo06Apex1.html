🚀 Creación de Archivos .h en C

Guía completa para crear tus propias bibliotecas de funciones

1

Archivo de Cabecera (.h)

Contiene las declaraciones de funciones y constantes

2

Archivo de Implementación (.c)

Implementa las funciones declaradas en el .h

3

Archivo Principal

Usa las funciones incluyendo el archivo .h

📁 Estructura de Archivos

proyecto/
├── main.c
├── mis_funciones.h
└── mis_funciones.c

1. Archivo de Cabecera (.h)

Este archivo contiene las declaraciones de funciones (prototipos) y constantes.

// mis_funciones.h
// Guardas de inclusión para evitar inclusiones múltiples
#ifndef MIS_FUNCIONES_H
#define MIS_FUNCIONES_H

// Incluir bibliotecas necesarias
#include <stdio.h>
#include <math.h>

// Constantes
#define PI 3.141592653589793
#define MAX_LONGITUD 100

// Declaraciones de funciones (prototipos)
int suma(int a, int b);
float calcular_promedio(float arr[], int n);
void imprimir_mensaje(char mensaje[]);
int es_primo(int numero);

#endif // MIS_FUNCIONES_H

2. Archivo de Implementación (.c)

Contiene las implementaciones reales de las funciones declaradas en el .h

// mis_funciones.c
#include "mis_funciones.h"

// Implementación de suma
int suma(int a, int b) {
    return a + b;
}

// Implementación de calcular_promedio
float calcular_promedio(float arr[], int n) {
    float total = 0.0;
    for(int i = 0; i < n; i++) {
        total += arr[i];
    }
    return total / n;
}

// Implementación de imprimir_mensaje
void imprimir_mensaje(char mensaje[]) {
    printf("Mensaje: %s\n", mensaje);
}

// Implementación de es_primo
int es_primo(int numero) {
    if (numero <= 1) return 0;
    for(int i = 2; i <= sqrt(numero); i++) {
        if (numero % i == 0) return 0;
    }
    return 1;
}

3. Archivo Principal (main.c)

Archivo que utiliza las funciones de tu biblioteca personalizada

// main.c
#include <stdio.h>
#include "mis_funciones.h"

int main(void) {
    // Usar funciones de tu biblioteca
    printf("Suma: %d\n", suma(5, 3));
    
    float numeros[] = {1.5, 2.5, 3.5, 4.5};
    printf("Promedio: %.2f\n", calcular_promedio(numeros, 4));
    
    imprimir_mensaje("¡Hola desde mi biblioteca!");
    
    printf("¿17 es primo? %s\n", es_primo(17) ? "Sí" : "No");
    
    return 0;
}

🔧 Compilación

# Compilar todos los archivos juntos
gcc -o programa main.c mis_funciones.c -lm

# O compilar por separado (recomendado para proyectos grandes)
gcc -c mis_funciones.c -o mis_funciones.o
gcc -c main.c -o main.o
gcc -o programa main.o mis_funciones.o -lm
💡 Tip: Usa -lm para enlazar la biblioteca matemática cuando uses funciones como sqrt(), pow(), etc.

⭐ Mejores Prácticas

✅ Usa Guardas de Inclusión

Siempre protege tus headers con #ifndef, #define, #endif para evitar inclusiones múltiples.

✅ Nombres Descriptivos

Usa nombres claros para funciones y constantes: calcular_area_circulo() en lugar de calc().

✅ Documenta tus Funciones

Incluye comentarios que expliquen qué hace cada función y sus parámetros.

✅ Organiza por Módulos

Agrupa funciones relacionadas en el mismo archivo .h (matemáticas, strings, archivos, etc.).

⚠️ Errores Comunes

❌ Olvidar las Guardas de Inclusión

Puede causar errores de redefinición de funciones.

❌ Definir Funciones en el .h

El archivo .h solo debe contener declaraciones, no implementaciones.

❌ No Incluir Dependencias

Si tus funciones usan printf, incluye stdio.h en el .h o .c según corresponda.
