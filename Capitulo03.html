Capítulo 3 – Desarrollo de programas estructurados en C

1. Algoritmos y control de programa

Definición 1 (Algoritmo). Procedimiento que especifica: 1. Las acciones a ejecutar.
2. El orden en que dichas acciones se llevan a cabo.

Definición 2 (Control de programa). Especificación del orden de ejecución de instrucciones en un programa.

Teorema de Böhm–Jacopini. Todo programa puede expresarse usando únicamente tres estructuras de control: - Secuencia - Selección - Repetición

2. Estructuras de control fundamentales

2.1 Secuencia

Las instrucciones se ejecutan una tras otra, en el orden escrito.

2.2 Selección

  • if (selección simple): 
    if (condición) instrucción;
  • if...else (selección doble): 
    if (condición) instrucción1; else instrucción2;
  • Operador condicional (?:): 
    (condición) ? expr1 : expr2;

    Único operador ternario en C.

2.3 Repetición

  • while: 
    while (condición) instrucción;

    Ejecuta instrucción mientras condición ≠ 0.

3. Pseudocódigo y metodología de diseño

Definición 3 (Pseudocódigo). Lenguaje informal, similar al inglés, que describe algoritmos sin depender de la sintaxis de un lenguaje de programación.

Metodología: Mejoramiento arriba-abajo, paso a paso

  1. Cima: descripción general del problema.
  2. Mejoramientos sucesivos: refinamiento en fases:
    • Inicialización
    • Procesamiento
    • Terminación

4. Variables especiales

Tipo Propósito Inicialización típica
Contador Cuenta número de iteraciones o entradas 0 o 1
Total Acumula suma de valores 0

Nota: No inicializar estas variables produce errores lógicos.

5. Tipos de repetición

Tipo Característica
Controlada por contador Número fijo de repeticiones conocido a priori.
Controlada por centinela Repetición termina al introducir un valor especial (e.g., -1).

6. Tipos de datos y operadores avanzados

6.1 Tipo float

Representa números de punto flotante. Para preservar fracción en promedios:

promedio = (float) total / contador;

6.2 Operadores de asignación compuesta

Operador Ejemplo Equivalente
+=x += 3;x = x + 3;
-=x -= 2;x = x - 2;
*=x *= 5;x = x * 5;
/=x /= 4;x = x / 4;
%=x %= 3;x = x % 3;

6.3 Operadores de incremento/decremento

Operador Ejemplo Comportamiento
++x preincremento Incrementa x, luego usa el nuevo valor.
x++ posincremento Usa valor actual de x, luego lo incrementa.
--x predecremento Decrementa x, luego usa el nuevo valor.
x-- posdecremento Usa valor actual, luego lo decrementa.

Restricción: Solo aplicable a nombres de variable simples.

8. Ejercicios resueltos representativos

Ejercicio R1: Promedio con repetición controlada por contador

int total = 0, contador = 1, calificacion;
while (contador <= 10) {
    scanf("%d", &calificacion);
    total += calificacion;
    contador++;
}
printf("Promedio: %d\n", total / 10);

Ejercicio R2: Promedio con centinela

int total = 0, contador = 0, calificacion;
scanf("%d", &calificacion);
while (calificacion != -1) {
    total += calificacion;
    contador++;
    scanf("%d", &calificacion);
}
if (contador != 0)
    printf("Promedio: %.2f\n", (float)total / contador);
else
    printf("No se introdujeron calificaciones\n");

Ejercicio R3: Análisis de resultados (aprobado/reprobado)

int aprobados = 0, reprobados = 0, estudiante = 1, resultado;
while (estudiante <= 10) {
    scanf("%d", &resultado);
    if (resultado == 1) aprobados++;
    else reprobados++;
    estudiante++;
}
printf("Aprobados: %d\nReprobados: %d\n", aprobados, reprobados);
if (aprobados > 8) printf("Objetivo alcanzado\n");
